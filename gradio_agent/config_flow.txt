  1️⃣  FUNCTION ARGUMENTS (Highest Priority)
       preload_vla_policy(policy_path="custom/path")
       ✅ Direct control
       ✅ Overrides everything

  2️⃣  YAML CONFIG FILE
       policy_config.yaml:
         policy_path: "my/policy"
       ✅ Team-friendly
       ✅ Version controlled
       ✅ Structured

  3️⃣  ENVIRONMENT VARIABLES
       export LEROBOT_POLICY_PATH="my/policy"
       OR
       .env file:
         LEROBOT_POLICY_PATH="my/policy"
       ✅ Production-ready
       ✅ CI/CD friendly

  4️⃣  DEFAULT VALUES (Lowest Priority)
       Hardcoded fallbacks
       ✅ Always works
       ✅ Quick start

┌─────────────────────────────────────────────────────────────────────┐
│  USAGE EXAMPLES                                                     │
└─────────────────────────────────────────────────────────────────────┘

  METHOD 1: YAML Config (Recommended)
  ────────────────────────────────────
    $ nano policy_config.yaml    # Edit settings
    $ python demo_tool_calling.py --preload-policy


  METHOD 2: .env File
  ────────────────────────────────────
    $ cp .env.example .env       # Create from template
    $ nano .env                  # Edit settings
    $ python demo_tool_calling.py --preload-policy


  METHOD 3: Environment Variables
  ────────────────────────────────────
    $ export LEROBOT_POLICY_PATH="my/policy"
    $ export LEROBOT_POLICY_TYPE="act"
    $ python demo_tool_calling.py --preload-policy


  METHOD 4: Programmatic
  ────────────────────────────────────
    from robot_tools import preload_vla_policy

    preload_vla_policy(
        policy_path="my/policy",
        policy_type="act"
    )

┌─────────────────────────────────────────────────────────────────────┐
│  QUICK TROUBLESHOOTING                                              │
└─────────────────────────────────────────────────────────────────────┘

  ❌ Config file not found
     → Check: ls -la gradio_agent/policy_config.yaml
     → Create from template if missing

  ❌ PyYAML not installed
     → Install: pip install pyyaml

  ❌ Wrong policy loaded
     → Check priority: Function args > YAML > Env > Defaults
     → Use: env | grep LEROBOT_ to see env vars

  ❌ GPU not available
     → Set device: "cpu" in config
     → Or: export LEROBOT_DEVICE="cpu"
